### 我如何新建一篇文章

我使用 Git Bash 新建文章

``` bash
$ hexo new "My New Post"
```

之后再本地运行测试

``` bash
$ hexo server
```

### 如何把文章上传到 GitHub

因为我网络的玄学原因，无法直接通过

```bash
$ hexo deploy
```

将文章更新推送到 GitHub

所以我直接把 public 文件夹里的内容上传到 GitHub 仓库
